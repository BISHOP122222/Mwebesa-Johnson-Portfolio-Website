# Attachments Directory

This directory contains downloadable resources for the portfolio.

## Contents

- **cv.pdf** - Professional CV/Resume
- **resume.pdf** - Resume document
- **screenshots/** - Screenshots of systems and projects

## Usage

Files in this directory are accessible via direct links:
- CV: `/attachments/cv.pdf`
- Resume: `/attachments/resume.pdf`
- Screenshots: `/attachments/screenshots/[filename]`

## Note

Place your CV, resume, and system screenshots in this directory for automatic integration with the portfolio.
